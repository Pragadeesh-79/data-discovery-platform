How DataShield Works

Four simple steps to discover, classify, and protect personal data across your enterprise.

FAQ

Frequently Asked Questions

DataShield is an enterprise-wide personal data discovery and classification platform aligned with India's Digital Personal Data Protection Act (DPDPA). It automatically scans your organization's documents, detects PII using regex and AI, classifies the data, and generates compliance reports.

The scanner engine supports PDF, CSV, Excel (.xlsx/.xls), and DOCX document formats. It can recursively scan local folders and extract text content from each file for PII detection.

DataShield detects Aadhaar numbers, PAN cards, phone numbers, email addresses via regex. The AI/NLP engine additionally detects person names, locations, and organizations using spaCy's Named Entity Recognition (NER) model.

Detected entities are automatically classified into three categories: Identifier (Aadhaar, PAN), Personal (names, locations), and Financial (bank details). Each record receives a risk score (High, Medium, Low) based on the sensitivity of the data type.

All discovered PII records are stored in a MongoDB database with full metadata including type, file location, department owner, sensitivity level, and timestamp. The backend exposes REST APIs (/scan-results, /inventory, /report) for data access.

Yes. DataShield generates DPDPA-aligned compliance reports with risk summaries, PII inventory details, source locations, and remediation recommendations. Reports can be exported for regulatory audits.
